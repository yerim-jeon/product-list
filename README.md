# Product List App

## Feature

- 상품 클릭시 상세 화면으로 이동합니다.
- 상세화면에서 디자인을 클릭하면 이미지를 확대하여 볼 수 있습니다.
- 뒷면 디자인 선택 시 디자인이 선택버튼이 활성화 됩니다.

## Stack

React, Styled-Components<br>

## Installation

```
npm install
npm start
```
